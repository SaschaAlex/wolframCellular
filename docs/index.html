Sprinklr

Sprinklr is a smart irrigation control system. It replaces your normal irrigation control system and make it smarter so you can have in real-time status about what’s happening. You can also use the app to start any sprinkler from anywhere you want. It will make you grass greener than your neighbours.

Major Features

  • Fail-free system to ensure it always completes or let you know if it doesn’t
  • Can use multiple boxes and is expandable
  • Web-based control (basically it works on anything that has the internet)
  • Adjust itself based on the weather conditions. (Weather API from where you choose)
  • Manual Control with a remote (On/Off Switch with timers and scheduler)
  • Scheduled Control Manager
  • Named Zones (so you can know where it’s happening)
  • Seasonal adjustment (so if you want it doesn’t start in winter)

Incomming Features

  • Webapp
  • Id organisation

OpenWeather / OpenWeatherMap

Just provide you apikey and lattidue,longitude (eg. "40.749748,-73.991618") of the sprinklers location and active it on the app.
